    re модуль: В Python для работы с регулярными выражениями используется модуль re.

    Шаблон: Это строка, которая содержит комбинацию символов и метасимволов, которые представляют собой шаблон для поиска.

    Метасимволы: Метасимволы - это специальные символы, которые имеют особое значение в регулярных выражениях. Например:
        . - соответствует любому одиночному символу, кроме новой строки.
        ^ - соответствует началу строки.
        $ - соответствует концу строки.
        * - соответствует нулю или более повторений предыдущего символа или группы символов.
        + - соответствует одному или более повторениям предыдущего символа или группы символов.
        ? - соответствует нулю или одному повторению предыдущего символа или группы символов.
        \ - экранирование специальных символов.

    Функции re модуля: Некоторые основные функции модуля re:
        re.search(pattern, string): Поиск первого соответствия шаблону в строке.
        re.match(pattern, string): Проверка, соответствует ли начало строки шаблону.
        re.split(pattern, string, maxsplit=0) Аналог str.split(), только разделение происходит по подстрокам, подходящим под шаблон pattern
        re.findall(pattern, string): Поиск всех совпадений шаблона в строке и возврат их в виде списка.
        re.sub(pattern, replacement, string): Замена всех совпадений шаблона в строке на указанную подстановку.

Для работы с файлами в Python есть встроенная функция open(), которая имеет следующие основные аргументы:

    file — строка, содержащая путь до файла. Путь может быть абсолютным (от буквы диска в Windows или от корневой папки в Linux) или относительным (от папки, в которой находится запущенная программа);
    mode — режим доступа к файлу. Рассмотрим следующие режимы:
        "r" — режим чтения текстового файла; является режимом по умолчанию;
        "w" — режим записи нового текстового файла; файл всегда создаётся заново;
        "a" — режим записи текстового файла; если файла нет, то он создаётся, иначе данные добавляются в конец существующего файла;
        "+" — добавляется в конец обозначения предыдущих режимов и позволяет работать с файлом одновременно в режиме чтения и записи.
    encoding — строка, содержащая обозначение кодировки текстового файла (например, "UTF-8").
